# angular-gulp
Angular build system using Gulp

### Requires [Node.js](https://nodejs.org/download/)

### Includes:
- Angular
- AngularUI Router
- Angular Cookies
- Bootstrap
- Font Awesome
- jQuery
- Browser refresh using Browsersync
- Browserify
- HTML,CSS and JS minification
- JSHint
- File Concat

### Creating a new app
```
$ cd /dir/where/downloaded/
$ npm install
$ gulp new --app yourAppName
$ cd apps/yourAppName
```

### Creating a build
```
$ cd /dir/where/downloaded/
$ cd apps/yourAppName
$ gulp build
```

### Lint JS
```
$ cd /dir/where/downloaded/
$ cd apps/yourAppName
$ gulp lint
```
